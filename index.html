Rockpro64 Home Server / NAS

I’ve had a RaspberryPi running as a home server for a while now, but as I’ve added new services the age of the little board has really begun to show. It’s simply not powerful enough to run everything I’d like it to: Pi-hole, Syncthing, Jellyfin, OpenVPN, and so on. I had been considering upgrading to RPi4, but stumbled across the RockPro64 Single Board Computer. With six cores, 4GB of RAM, and a solid NAS case this seemed perfect for my needs.
